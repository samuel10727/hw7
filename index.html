重力測站數據計算與分析

1. 原始數據 (Given Data)

  • 緯度 (φ): 48.1195° N
  • 高度 (h): 487.9 m
  • 觀測重力 (gobs): 980,717.39 mGal
  • 假設地殼密度 (ρ): 2.67 g/cm³ (2670 kg/m³)
  • 海水密度 (ρw): 1.03 g/cm³

2. 問題 (a) 計算步驟

i) 理論重力 (Theoretical Gravity, gth):
使用 1967 國際重力公式：
gth = 978031.85 (1 + 0.00527889 sin²φ + 0.00002346 sin&sup4;φ) mGal
代入 φ = 48.1195°:
gth ≈ 980,904.34 mGal
ii) 自由空氣修正 (Free-Air Correction, FAC):
修正公式：FAC = 0.3086 × h
FAC = 0.3086 × 487.9 = 150.57 mGal
iii) 布格修正 (Bouger Correction, BC):
修正公式：BC = 0.04193 × ρ × h
BC = 0.04193 × 2.67 × 487.9 = 54.62 mGal
iv) 自由空氣重力異常 (Free-Air Anomaly, ΔgFA):
公式：ΔgFA = gobs + FAC - gth
ΔgFA = 980,717.39 + 150.57 - 980,904.34 = -36.38 mGal
v) 布格重力異常 (Bouger Anomaly, ΔgB):
公式：ΔgB = ΔgFA - BC
ΔgB = -36.38 - 54.62 = -91.00 mGal

3. 問題 (b) 海洋環境重計算

當觀測站位於海平面 (h=0)，但下方有 487.9m 深的水層時：

  • FAC: 因為觀測點就在海平面，h=0，故 FAC = 0。
  • ΔgFA: gobs - gth = 980,717.39 - 980,904.34 = -186.95 mGal。
  • 海洋布格修正: 需要補償水與地殼岩石之間的密度差異 (ρ - ρw)。
    BCocean = 0.04193 × (2.67 - 1.03) × 487.9 = 33.55 mGal。
  • ΔgB (Ocean): ΔgFA + BCocean = -186.95 + 33.55 = -153.40 mGal。
